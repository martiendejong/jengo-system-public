# jengo-system-public

Shareable capabilities, public tools and skills.

## What's Here

| Path | Purpose |
|------|---------|
| `skills/*.md` | Public skill definitions for reuse |
| `tools/*.ps1` | Open-source PowerShell tools and automation scripts |

## Architecture

This repo is part of the **Jengo 3-Tier Modular System**:
- **Layer:** system (how Jengo operates — skills, tools, policies, engine)
- **Tier:** public (safe to share, publish, or include in open-source)

Public system capabilities that are useful to other agents or developers. Internal policies and sensitive tools remain in `jengo-system-private`.
